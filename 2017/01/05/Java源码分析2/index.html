Vector 是 ArrayList 的线程安全版本，依靠synchronized实现，其他大多一样，本文主要说明两者不同的地方

成员变量不同

可以看出 Vector 多了一个capacityIncrement变量，暂且翻译为容量增量

构造方法不同

初始化调用默认构造方法时，其会默认容量为10然后调用一个参数的构造方法，当调用一个参数的构造方法时默认容量增量为0

添加方法

除了多了个synchronized关键字其它基本一样

扩容方法有差异

总结

Vector依靠synchronized关键字实现线程安全的 ArrayList，但是这样会造成效率比较低。默认容量就为10，容量增量为0.数组扩容时，容量增量大于0则新容量为旧容量与容量增量的和，否则为旧容量大小的两倍。
