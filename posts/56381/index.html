Spring Boot 在 IDEA 中使用 Lombok

字数统计 1,000 | 阅读时长 6

Lombok 是一个帮助程序员简化代码的一个库。当你对 JavaBean 中个某个属性进行改动时，你需要频繁的去改动 Getter 和 Setter 或 toString 或equals等方法，
当你使用 Lombok，只需一个@Data就能帮你自动生成以上的方法。详情点击官方网站，github 地址为https://github.com/rzwitserloot/lombok

安装插件

在 IDEA 中使用Lombok需要安装插件

  • Windows 操作系统：

    • File > Settings > Plugins > Browse repositories… > Search for “lombok” > Install Plugin

  • MacOs 操作系统:

    • Preferences > Settings > Plugins > Browse repositories… > Search for “lombok” > Install Plugin
  • 通常的方法:
    • 下载 latest release 然后安装它 Preferences > Plugins > Install plugin from disk…

安装完后重启 IDEA

IntelliJ IDEA 的设置

点击 Preferences -> "Build, Execution, Deployment -> "Compiler, Annotation Processors". 点击 Enable Annotation Processing

然后你可能需要全部重新编译你的项目通过"Rebuild Project".
